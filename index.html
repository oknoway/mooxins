mooxins: CSS3 mixins for SASS(SCSS)

mooxins generate vendor prefixes for CSS3 properties.

CSS properies:

  • opacity
  • border-radius
  • box-shadow
  • text-shadow
  • box-sizing
  • background-clip
  • text-overflow
  • border-image
  • transform
  • linear-gradient
  • radial-gradient
  • transition
  • transition-property
  • transition-duration
  • transition-timing-function
  • transition-delay

Also quick two type of gradients:

  • gradient-horizontal(startColor, endColor)
  • gradient-vertical(startColor, endColor)

Getting started

  1. Install SASS
  2. Import mooxins: @import 'mooxins';
  3. Include it: .my-gradient { @include gradient-vertical(#ff00c6, #a90083); }
